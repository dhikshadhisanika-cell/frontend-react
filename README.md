# Frontend React Project

## Description
This is a simple React frontend project created as part of a frontend developer internship task. It includes multiple pages, reusable components, routing, and responsive design.

## Tech Stack
- React
- Vite
- JavaScript
- HTML
- CSS
- React Router

## Run Instructions

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Build the project

```bash
npm run build
```
